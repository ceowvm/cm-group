# CM Group Mini App — демоверсия

Реализовано:
- стартовый экран;
- квиз из 5 вопросов;
- персональная диагностика;
- сбор имени и телефона;
- экран подарков;
- личный кабинет;
- материалы и продукты;
- финансовый калькулятор;
- Telegram WebApp API и `sendData()`;
- сохранение прогресса через localStorage.

## Запуск
Откройте `index.html` в браузере.

## Публикация
Загрузите папку на Netlify Drop или другой HTTPS-хостинг.

## SaleBot
Создайте кнопку Web App и укажите HTTPS-адрес приложения. Данные отправляются через `Telegram.WebApp.sendData()`.

Реальные ссылки на видео, книгу, тарифы и оплату пока заменены демо-действиями.
